# Vistral

A toolbox for your clean architecture application.

## Disclaimer

Please note that this library is in the early stages of development and is
subject to major changes. Use at your own risk.

## Installation

Install via `pip`:

```shell
pip install vistral
```

Or via `poetry`:

```shell
poetry add vistral
```
